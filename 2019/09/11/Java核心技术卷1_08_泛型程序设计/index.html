Java 泛型转换的事实：

  • 虚拟机中没有泛型，只有普通的类和方法。
  • 所有的类型参数都用它们的限定类型替换。
  • 桥方法被合成来保持多态。
  • 为保持类型安全性，必要时插人强制类型转换。
